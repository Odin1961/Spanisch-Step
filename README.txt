ENGLISH STEP – INSTALLATION

Dieses Paket ist eine installierbare Web-App (PWA).

Wichtig:
Eine PWA muss über eine HTTPS-Webadresse geöffnet werden. Das bloße Öffnen
der index.html-Datei aus der Dateien-App reicht für Offline-Modus und echte
Home-Bildschirm-Installation nicht zuverlässig aus.

Einfachste Veröffentlichung:
1. Den Inhalt dieses ZIP-Pakets bei einem statischen Webhoster hochladen
   (zum Beispiel GitHub Pages, Netlify oder Cloudflare Pages).
2. Die dadurch erzeugte HTTPS-Adresse auf dem iPad in Safari öffnen.
3. Teilen > Zum Home-Bildschirm > „Als Web-App öffnen“ aktivieren > Hinzufügen.

Enthalten:
- Gesprächssituationen ohne API
- deutsche Untertitel ein/aus
- regelbare Sprechgeschwindigkeit
- Spracheingabe, sofern Safari sie bereitstellt
- Offline-Cache nach dem ersten Laden
- lokaler Fortschritt
